Youdao ဖန်သားပြင်ဖမ်းယူ / ရွေးချယ်ဘာသာပြန် / OCR / ဖြတ်လမ်း - ဖြေရှင်းနည်းရှာဖွေ

Youdao ၏ ဖန်သားပြင်ပေါ်စကားလုံးဖမ်းယူ (hover ဖြင့်ရှာ), ရွေးချယ်ဘာသာပြန်, OCR သို့မဟုတ် ဖြတ်လမ်း ရုတ်တရက်ရပ်သွားသလား? အရင်ဆုံး သုံးခုရှင်းပါ: မဖွင့်ထား / ခွင့်မရှိ, app တစ်ခု (Office / PDF / browser) ၏ တွဲဖက်နိုင်မှု, သို့မဟုတ် အကြောင်းအရာကိုယ်တိုင် (scan / hardware acceleration) ဖတ်နိုင်သောစာသားမရှိ? သင်ကြုံတွေ့သော လက္ခဏာ သို့မဟုတ် keyword ရှာပါ (ဥပမာ ဖမ်းယူ, pdf, ရွေးချယ်, ဖြတ်လမ်း, 64-bit, OCR): အရင်ဆုံး L1 လက္ခဏာနှင့် တရားဝင်နေရာ, ထို့နောက် L2 အရင်းခံအကြောင်းရင်းနှင့် ဖြေရှင်းနည်း, နောက်ဆုံး ပြိုင်ဘက်နှိုင်းယှဉ်မှုနှင့် တကယ့်အမှုများ. ထည့်သွင်း / တပ်ဆင်? ကြည့်ပါ Youdao ဒေါင်းလုဒ် / ဖမ်းယူနှင့် ရွေးချယ်လမ်းညွှန်.

အဆင့် 1: အရင်ဆုံး လေးအုပ်စုခွဲပါ (L1 - ချက်ချင်းပြန်မထည့်ပါနှင့်)

Youdao ဖမ်းယူ/ရွေးချယ် ရပ်သောအခါ, 10 ကြိမ်တွင် 9 ကြိမ်က လေးအုပ်စုထဲမှ တစ်ခုဖြစ်သည် - အရင်ဆုံး အပြစ်ရှာပါ: ① ဖွင့်ခြင်း/setting — Youdao background တွင်မလည်ပတ် (tray icon မီးခိုး), သို့မဟုတ် «ဖမ်းယူ / ရွေးချယ် / ရွေးချယ်စဉ်အလိုအလျောက်ဘာသာပြန် ဖွင့်» မ check, သို့မဟုတ် trigger မှား; ② ခွင့်ပြုချက်များ — macOS «Accessibility + Screen Recording» မပေး, သို့မဟုတ် Windows တွင် target က elevated app ဖြစ်ပြီး Youdao administrator အဖြစ်မလည်ပတ်; ③ app တွဲဖက်နိုင်မှု — Word/Excel တွင် Youdao COM add-in ပိတ်, Acrobat protected mode injection ပိတ်ဆို့, သို့မဟုတ် browser တွင် ဖမ်းယူ extension မရှိ; ④ rendering/အကြောင်းအရာ — scan PDF က ပုံ (OCR လို), browser/PDF က GPU hardware acceleration ဖြင့် စာသားကို texture အဖြစ်ဆွဲ, သို့မဟုတ် မကိုက်ညီသော multi-monitor DPI က popup ကို ဖန်သားပြင်ပြင်ပ ပစ်ထုတ်. အပြည့်အစုံ setting: Youdao ဖမ်းယူနှင့် ရွေးချယ်လမ်းညွှန်.

ဖွင့်ခြင်း / settingဖမ်းယူ · ရွေးချယ် · အလိုအလျောက်ဘာသာပြန် · OCR ပါဝါ
Youdao background တွင်လည်ပတ်ကြောင်း သေချာပါ (tray icon မီးခိုးမဟုတ်), setting → ဖမ်းယူနှင့် ရွေးချယ် သို့သွားပြီး «ဖမ်းယူဖွင့်», «ရွေးချယ်ဖွင့်», «ရွေးချယ်စဉ်အလိုအလျောက်ဘာသာပြန်» check လုပ်ပြီး မှန်ကန်သော trigger ရွေးပါ (hover / Ctrl ဖိ).
ခွင့်ပြုချက်များmacOS Accessibility+Screen Recording · Windows administrator/UAC
macOS: System Settings → Privacy & Security, Youdao ကို «Accessibility» နှင့် «Screen Recording» ပေးပါ; Windows: target က «administrator အဖြစ်လည်ပတ်» သောအခါ, Youdao ကိုလည်း administrator အဖြစ်လည်ပတ်ပါ (hook က အခွင့်ထူးအဆင့်တူတွင်သာ inject).
app တွဲဖက်နိုင်မှုOffice COM · Acrobat protected mode · 64-bit · browser extension
Office: File → Options → Add-ins → COM add-ins, Youdao add-in check; Acrobat: «startup တွင် protected mode ဖွင့်» ပိတ်, လိုအပ်လျှင် YodaoDict.api ကို plug_ins သို့ copy; browser: Youdao ဖမ်းယူ extension ထည့်.
ဖြတ်လမ်းပဋိပက္ခ · စိတ်ကြိုက် · OCR screenshot key
setting → ဖြတ်လမ်း, ပဋိပက္ခသတိပေးချက်ကြည့်; Ctrl+Shift / Alt+စာလုံး ကဲ့သို့ ပဋိပက္ခနည်းသော ပေါင်းစပ်သို့ပြောင်း; OCR screenshot ကို သီးသန့် key ပေး (ဥပမာ Ctrl+Alt+O, အခြား app များမှ ဝေး).

GPU hardware acceleration ပါသော browser/PDF က အကြောင်းအရာကို ဖတ်နိုင်သောစာသားမပါဘဲ texture render (acceleration ပိတ်); scan PDF က ပုံ (OCR သုံး); multi-monitor တွင် ဖန်သားပြင်ပြင်ပ popup အတွက် «window position pin» check.
network / installationဖမ်းယူအမြန်နှုန်း · portable ဗားရှင်း · add-in မ load
online ရှာဖွေ နှေး? network စစ်; တရားဝင် full ဗားရှင်းအမြဲထည့်ပါ - တရားဝင်မဟုတ်သော «portable/ပေါ့ပါး» ဗားရှင်းများတွင် register လုပ်ထားသော ဖမ်းယူ-hook DLL မရှိ, ထို့ကြောင့် PDF/global ဖမ်းယူ ချက်ချင်းပျက်.

အဆင့် 2: အလျစ်လျူရှုဆုံး အရင်းခံအကြောင်းရင်း → L2 (ဖန်သားပြင်ဖမ်းယူ = global hook / DLL injection)

အဓိကအချက်: ဖန်သားပြင်ဖမ်းယူသည် «ဖန်သားပြင်ပုံကို မဖတ်» - Youdao က ဖမ်းယူ code (global hook / DLL) ကို အခြား app ထဲ inject လုပ်ပြီး ၎င်း၏ text buffer ကိုဖတ်သည်. ၎င်းသည် «မရှင်းနိုင်သော» ချို့ယွင်းမှုအများစုကို ရှင်းပြသည်: ① အခွင့်ထူးအဆင့် — ပုံမှန်အခွင့်အရေး Youdao သည် «administrator အဖြစ်လည်ပတ်» app ထဲ inject မလုပ်နိုင် (ဆန့်ကျင်လည်း), ထို့ကြောင့် «ထို app သာ မဖမ်းယူ» သည် များသောအားဖြင့် မကိုက်ညီသောအခွင့်ထူး; ② bitness — 32/64-bit hook များ မရောစပ်, ထို့ကြောင့် ဟောင်းသော Youdao သည် 64-bit Acrobat တွင် «မတွဲဖက်နိုင်သော plug-in, YodaoDict.api load မလုပ်နိုင်» ပေးသည်; ③ sandbox — Acrobat ၏ «protected mode» သည် third-party .api injection ကိုပိတ်ဆို့သော sandbox; ④ hardware acceleration — browser/PDF က အကြောင်းအရာကို GPU texture အဖြစ် render လုပ်သောအခါ ဖမ်းယူရန် «ဖတ်နိုင်သောစာသား» မကျန်; ⑤ scan — ပုံသက်သက် PDF တွင် text layer မရှိ, «OCR ပါဝါဖမ်းယူ» သုံးပါ. အောက်တွင် self-check panel; ထို့နောက် ပြိုင်ဘက်နှိုင်းယှဉ်မှုနှင့် 2026 ခန့်မှန်းချက်များ.

တကယ့်အမှုများ - အမြန်ဖတ်

  • browser တွင်မဖမ်းယူ ဒါပေမဲ့ Notepad / Word ပုံမှန်: များသောအားဖြင့် browser တွင် Youdao ဖမ်းယူ extension မရှိ, သို့မဟုတ် GPU hardware acceleration က စာမျက်နှာကို texture render → extension ထည့်, သို့မဟုတ် «hardware acceleration သုံး» ပိတ်ပြီး ပြန်စ.
  • Acrobat 64-bit, PDF တွင် ရွေးချယ်ခြင်း ဘာမှမလုပ်, «မတွဲဖက်နိုင်သော plug-in, YodaoDict.api load မလုပ်နိုင်»: Acrobat တွင် Edit → Preferences → Security (Enhanced) → «startup တွင် protected mode ဖွင့်» ပိတ်, Acrobat မတိုင်မီ Youdao ဖွင့်; ဆက်ဖြစ်လျှင် YodaoDict.api ကို Youdao folder မှ Acrobat ၏ plug_ins သို့ copy.
  • PDF စာသား ရွေးနိုင် ဒါပေမဲ့ မဖမ်းယူ, သို့မဟုတ် စာမျက်နှာတစ်ခုလုံးမရ: ၎င်းသည် scan (text layer မပါသော ပုံ PDF, «OCR ပါဝါဖမ်းယူ» လို), သို့မဟုတ် Acrobat protected mode / Autotray.exe lock က injection ပိတ်ဆို့.
  • Office (အထူးသဖြင့် Education / 365) ထည့်ပြီးနောက် Word က ရွေးချယ်မှု မဘာသာပြန်: Youdao Office COM add-in ပိတ် → File → Options → Add-ins → COM add-ins → ဖွင့်; မဟုတ်လျှင် ပြန်register ရန် Youdao ပြန်ထည့်.
  • OS update ပြီးနောက် macOS တွင် ဖမ်းယူနှင့် ဖြတ်လမ်း သေ: update က ခွင့်ပြုချက်များ reset → Privacy & Security တွင် Youdao ကို «Accessibility» နှင့် «Screen Recording» ပြန်ပေးပြီး ပြန်စ.
  • ပုံမှန် app များတွင် ဖမ်းယူအလုပ်လုပ် ဒါပေမဲ့ «administrator အဖြစ်လည်ပတ်» app တွင်ပျက်: မတူသော အခွင့်ထူးအဆင့်များက global hook ကို elevated process ထဲ inject ခြင်းကို ပိတ်ဆို့ → Youdao ကို right-click «Run as administrator».
  • multi-monitor / high DPI: ဖမ်းယူ popup က မမြင်ရသောနေရာသို့ «ပျံ»: per-screen DPI scaling မကိုက်ညီခြင်းကြောင့် → ဖမ်းယူ setting တွင် «window position pin» check ပြီး main screen ၏ ဘယ်ဘက်အပေါ်ထောင့်တွင် ပင်.
