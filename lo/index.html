ການຈັບຄຳໃນໜ້າຈໍ ຂອງ Youdao (ຄົ້ນຫາໂດຍວາງເມົ້າ), ການແປຕາມການເລືອກ, OCR ຫຼື ທາງລັດ ຢຸດທັນທີບໍ? ກ່ອນອື່ນຊີ້ແຈງສາມເລື່ອງ: ບໍ່ໄດ້ເປີດໃຊ້ / ບໍ່ມີສິດ, ຄວາມເຂົ້າກັນໄດ້ຂອງແອັບໃດໜຶ່ງ (Office / PDF / ບຣາວເຊີ), ຫຼື ເນື້ອໃນເອງ (ສະແກນ / ການເລັ່ງຮາດແວ) ບໍ່ມີຂໍ້ຄວາມທີ່ອ່ານໄດ້? ຄົ້ນຫາ ອາການ ຫຼື ຄຳສຳຄັນ ທີ່ທ່ານພົບ (ເຊັ່ນ ການຈັບ, pdf, ການເລືອກ, ທາງລັດ, 64-ບິດ, OCR): ກ່ອນອື່ນ ອາການ L1 ແລະ ຕຳແໜ່ງທາງການ, ຈາກນັ້ນ ສາເຫດຮາກ L2 ແລະ ວິທີແກ້, ສຸດທ້າຍການປຽບທຽບຄູ່ແຂ່ງ ແລະ ກໍລະນີຈິງ. ຕິດຕັ້ງ / ຕັ້ງຄ່າ? ເບິ່ງ ດາວໂຫຼດ Youdao / ຄູ່ມືການຈັບ ແລະ ການເລືອກ.

ເມື່ອການຈັບ/ການເລືອກ Youdao ຢຸດ, 9 ໃນ 10 ເທື່ອມັນແມ່ນໜຶ່ງໃນສີ່ກຸ່ມ - ຊອກຫາຄວາມຜິດກ່ອນ: ① ການເປີດໃຊ້/ການຕັ້ງຄ່າ — Youdao ບໍ່ແລ່ນໃນພື້ນຫຼັງ (ໄອຄອນ tray ສີຂີ້ເຖົ່າ), ຫຼື ບໍ່ໄດ້ໝາຍ «ເປີດໃຊ້ການຈັບ / ການເລືອກ / ແປອັດຕະໂນມັດເມື່ອເລືອກ», ຫຼື ຕົວກະຕຸ້ນຜິດ; ② ສິດ — macOS «ການເຂົ້າເຖິງ + ການບັນທຶກໜ້າຈໍ» ບໍ່ໄດ້ໃຫ້, ຫຼື ໃນ Windows ເປົ້າໝາຍແມ່ນແອັບສິດສູງ ແຕ່ Youdao ບໍ່ແລ່ນເປັນຜູ້ບໍລິຫານ; ③ ຄວາມເຂົ້າກັນໄດ້ຂອງແອັບ — Youdao COM add-in ໃນ Word/Excel ຖືກປິດ, Acrobat protected mode ກີດຂວາງການສັກ, ຫຼື ບຣາວເຊີບໍ່ມີສ່ວນຂະຫຍາຍການຈັບ; ④ ການເຣນເດີ/ເນື້ອໃນ — ສະແກນ PDF ເປັນຮູບ (ຕ້ອງ OCR), ບຣາວເຊີ/PDF ກັບການເລັ່ງຮາດແວ GPU ແຕ້ມຂໍ້ຄວາມເປັນ texture, ຫຼື DPI ຫຼາຍຈໍບໍ່ກົງ ໂຍນປ໊ອບອັບອອກນອກໜ້າຈໍ. ການຕັ້ງຄ່າເຕັມ: ຄູ່ມືການຈັບ ແລະ ການເລືອກ Youdao.

ການເປີດໃຊ້ / ການຕັ້ງຄ່າການຈັບ · ການເລືອກ · ແປອັດຕະໂນມັດ · OCR ແຮງ
ໃຫ້ແນ່ໃຈວ່າ Youdao ແລ່ນໃນພື້ນຫຼັງ (ໄອຄອນ tray ບໍ່ແມ່ນສີຂີ້ເຖົ່າ), ໄປທີ່ ການຕັ້ງຄ່າ → ການຈັບ ແລະ ການເລືອກ ແລ້ວໝາຍ «ເປີດໃຊ້ການຈັບ», «ເປີດໃຊ້ການເລືອກ», «ແປອັດຕະໂນມັດເມື່ອເລືອກ» ແລະ ເລືອກຕົວກະຕຸ້ນທີ່ຖືກຕ້ອງ (ວາງເມົ້າ / ກົດ Ctrl).

ການເຣນເດີ / ການສະແດງການເລັ່ງຮາດແວ · ສະແກນ · DPI ຫຼາຍຈໍ · ເຕັມຈໍ

ກໍລະນີຈິງ - ອ່ານໄວ

  • ບໍ່ຈັບໃນບຣາວເຊີ ແຕ່ Notepad / Word ປົກກະຕິ: ປົກກະຕິບຣາວເຊີບໍ່ມີສ່ວນຂະຫຍາຍການຈັບ Youdao, ຫຼື ການເລັ່ງຮາດແວ GPU ເຣນເດີໜ້າເປັນ texture → ຕິດຕັ້ງສ່ວນຂະຫຍາຍ, ຫຼື ປິດ «ໃຊ້ການເລັ່ງຮາດແວ» ແລ້ວເລີ່ມໃໝ່.
  • Acrobat 64-ບິດ, ການເລືອກໃນ PDF ບໍ່ມີຫຍັງ, «ປລັກອິນບໍ່ເຂົ້າກັນ, ໂຫຼດ YodaoDict.api ບໍ່ໄດ້»: ໃນ Acrobat ແກ້ໄຂ → ການຕັ້ງຄ່າ → ຄວາມປອດໄພ (ຂັ້ນສູງ) → «ເປີດໃຊ້ protected mode ເມື່ອເລີ່ມ» ປິດ, ເປີດ Youdao ກ່ອນ Acrobat; ຖ້າຍັງ copy YodaoDict.api ຈາກໂຟນເດີ Youdao ໄປ plug_ins ຂອງ Acrobat.
  • ຂໍ້ຄວາມ PDF ເລືອກໄດ້ ແຕ່ບໍ່ຈັບ, ຫຼື ໜ້າທັງໝົດບໍ່ໄດ້: ມັນເປັນສະແກນ (ຮູບ PDF ບໍ່ມີຊັ້ນຂໍ້ຄວາມ, ຕ້ອງ «OCR ການຈັບແບບແຮງ»), ຫຼື Acrobat protected mode / ການລັອກ Autotray.exe ກີດຂວາງການສັກ.
  • ຫຼັງຕິດຕັ້ງ Office (ໂດຍສະເພາະ Education / 365) Word ບໍ່ແປການເລືອກ: Youdao Office COM add-in ຖືກປິດ → ໄຟລ໌ → ຕົວເລືອກ → Add-in → COM add-in → ເປີດໃຊ້; ບໍ່ດັ່ງນັ້ນຕິດຕັ້ງ Youdao ໃໝ່ເພື່ອລົງທະບຽນຄືນ.
  • ຫຼັງອັບເດດ OS ການຈັບ ແລະ ທາງລັດ ຕາຍໃນ macOS: ອັບເດດ reset ສິດ → ໃຫ້ Youdao «ການເຂົ້າເຖິງ» ແລະ «ການບັນທຶກໜ້າຈໍ» ຄືນໃນ ຄວາມເປັນສ່ວນຕົວ ແລະ ຄວາມປອດໄພ ແລ້ວເລີ່ມໃໝ່.
  • ການຈັບເຮັດວຽກໃນແອັບປົກກະຕິ ແຕ່ລົ້ມເຫຼວໃນແອັບ «ແລ່ນເປັນຜູ້ບໍລິຫານ»: ລະດັບສິດຕ່າງກັນກີດຂວາງ global hook ສັກເຂົ້າ process ສິດສູງ → ຄລິກຂວາ Youdao «ແລ່ນເປັນຜູ້ບໍລິຫານ».
  • ຫຼາຍຈໍ / DPI ສູງ: ປ໊ອບອັບການຈັບ «ບິນ» ໄປບ່ອນທີ່ເບິ່ງບໍ່ເຫັນ: ຍ້ອນການ scale DPI ແຕ່ລະຈໍບໍ່ກົງ → ໝາຍ «ປັກໝຸດຕຳແໜ່ງໜ້າຕ່າງ» ໃນການຕັ້ງຄ່າການຈັບ ເພື່ອປັກໄວ້ມຸມເທິງຊ້າຍຂອງຈໍຫຼັກ.
