Youdao 사전 화면 단어캡처 / 선택 번역 / 스크린샷 OCR / 단축키 해결 검색

Youdao 사전의 화면 단어캡처(마우스를 올려 조회), 선택 번역, 스크린샷 OCR 또는 단축키가 갑자기 안 되나요? 먼저 세 가지를 구분하세요 — 활성화/권한 문제인지, 특정 앱(Office / PDF / 브라우저)의 호환성인지, 콘텐츠 자체(스캔본 / 하드웨어 가속)에 읽을 수 있는 텍스트가 없는지. 겪고 있는 증상이나 키워드(예: 캡처, pdf, 선택, 단축키, 64비트, 스크린샷)를 검색해 먼저 L1 증상·공식 위치를, 다음으로 L2 근본원인·해결을, 마지막으로 경쟁 제품 비교와 실제 사례를 보세요. 설치/설정이 필요하면 Youdao 사전 다운로드 / 캡처·선택 설정 가이드로.

1단계: 먼저 네 가지로 분류 (L1·바로 재설치하지 말 것)

Youdao 캡처/선택이 안 될 때 열에 아홉은 네 가지 중 하나입니다 — 누구의 문제인지 먼저 가리세요: ① 활성화/설정 — Youdao가 백그라운드에서 안 돎(트레이 아이콘 회색), 또는 「캡처/선택/선택 시 자동 번역」을 안 켰거나 캡처 방식이 틀림; ② 권한 — macOS에서 「손쉬운 사용 + 화면 기록」을 허용하지 않음, Windows에서 대상이 관리자 권한 앱인데 Youdao는 관리자로 실행 안 함; ③ 앱 호환 — Word/Excel의 Youdao COM 추가 기능이 사용 안 함, Acrobat 보호 모드가 인젝션 차단, 브라우저에 캡처 확장 없음; ④ 렌더링/콘텐츠 — 스캔 PDF는 이미지(OCR 필요), 브라우저/PDF가 GPU 하드웨어 가속으로 글자를 텍스처화, 멀티모니터 DPI 불일치로 팝업이 화면 밖. 자세한 설정은 Youdao 캡처·선택 설정 가이드.

활성화 / 설정캡처 · 선택 · 자동 번역 · OCR 강력 캡처
먼저 Youdao가 백그라운드에서 실행 중인지(트레이 아이콘 비회색) 확인하고, 설정 → 캡처·선택에서 「캡처 사용」「선택 사용」「선택 시 자동 번역」을 체크하고 올바른 캡처 방식(마우스오버 / Ctrl)을 선택.
권한macOS 손쉬운 사용+화면 기록 · Windows 관리자/UAC
macOS: 시스템 설정 → 개인정보 보호 및 보안에서 Youdao에 「손쉬운 사용」「화면 기록」 허용; Windows: 대상이 「관리자로 실행」이면 Youdao도 관리자로 실행(같은 권한 레벨이어야 후크가 주입됨).
앱 호환Office COM · Acrobat 보호 모드 · 64비트 · 브라우저 확장
Office: 파일 → 옵션 → 추가 기능 → COM 추가 기능에서 Youdao 사용; Acrobat: 「시작 시 보호 모드 사용」 끄고 필요시 YodaoDict.api를 plug_ins에 복사; 브라우저: Youdao 캡처 확장 설치.
단축키충돌 · 사용자 지정 · 스크린샷 OCR 키
설정 → 단축키에서 충돌 알림 확인; Ctrl+Shift / Alt+문자처럼 덜 겹치는 조합으로 변경; 스크린샷 OCR은 별도 키(예: Ctrl+Alt+O, 다른 앱과 회피).
렌더링 / 표시하드웨어 가속 · 스캔본 · DPI 멀티모니터 · 전체 화면
브라우저/PDF가 GPU 하드웨어 가속으로 콘텐츠를 텍스처화하면 가시 영역에 읽을 텍스트 없음(가속 끄기); 스캔 PDF는 이미지라 OCR 필요; 멀티모니터 팝업이 날아가면 「창 위치 고정」 체크.
네트워크 / 설치캡처 속도 · 포터블 빌드 · 추가 기능 미로드
온라인 조회가 느리면 네트워크 확인; 반드시 공식 정식판 설치 — 비공식 「포터블/경량판」은 등록된 캡처 후크 DLL이 없어 PDF/글로벌 캡처가 바로 실패.

2단계: 가장 간과되는 근본원인 → L2 심층 (화면 캡처 = 글로벌 후크 / DLL 인젝션)

핵심: 화면 캡처는 「화면 이미지를 읽는」 게 아니라, Youdao가 상대 앱에 캡처 코드(글로벌 후크 / DLL)를 주입해 그 텍스트 버퍼를 읽는 것입니다. 이걸 알면 「알 수 없는」 실패의 대부분이 설명됩니다 — ① 권한 레벨: 일반 권한의 Youdao는 「관리자로 실행」된 앱에 주입할 수 없고(반대도) 그래서 「그 앱만 캡처 안 됨」은 보통 권한 불일치; ② 비트수: 32/64비트 후크는 호환 안 되어 구버전 Youdao는 64비트 Acrobat에서 「호환되지 않는 플러그인, YodaoDict.api를 로드할 수 없음」; ③ 샌드박스: Acrobat 「보호 모드」는 샌드박스로 제3자 .api 주입을 차단; ④ 하드웨어 가속: 브라우저/PDF가 GPU로 콘텐츠를 텍스처화하면 가시 영역에 「읽을 수 있는 텍스트」가 없어 캡처 불가; ⑤ 스캔본: 이미지 전용 PDF는 텍스트 레이어가 없어 「OCR 강력 캡처」가 필요. 아래는 캡처 활성화 자가점검 패널, 그 다음은 경쟁 제품 비교와 2026 추정입니다.

실제 사례 빠른 판단

  • 브라우저에서 캡처가 안 되는데 메모장/Word는 정상: 보통 브라우저에 Youdao 캡처 확장이 없거나 GPU 하드웨어 가속으로 페이지가 텍스처화됨 → 확장 설치, 또는 브라우저에서 「하드웨어 가속 사용」 끄고 재시작.
  • Acrobat 64비트, PDF 선택 무반응, 「호환되지 않는 플러그인, YodaoDict.api 로드 불가」: Acrobat의 편집 → 기본 설정 → 보안(고급) → 「시작 시 보호 모드 사용」 끄고 Youdao를 먼저, Acrobat을 나중에 실행; 그래도 안 되면 Youdao 폴더의 YodaoDict.api를 Acrobat plug_ins에 복사.
  • PDF 텍스트는 선택되는데 캡처가 안 되거나 페이지 전체가 안 됨: 스캔본(이미지 PDF, 텍스트 레이어 없음, 「OCR 강력 캡처」 필요)이거나 Acrobat 보호 모드 / Autotray.exe 점유로 주입 차단.
  • Office(특히 교육용/365) 설치 후 Word에서 선택 번역 안 됨: Youdao Office COM 추가 기능이 사용 안 함 → 파일 → 옵션 → 추가 기능 → COM 추가 기능 → 사용; 안 되면 Youdao 재설치로 추가 기능 재등록.
  • OS 업데이트 후 macOS에서 캡처·단축키 전부 죽음: 업데이트가 권한을 초기화 → 개인정보 보호 및 보안에서 Youdao에 「손쉬운 사용」「화면 기록」 재허용 후 재시작.
  • 일반 앱에선 캡처되는데 어떤 「관리자로 실행」 앱에선 안 됨: 권한 레벨이 달라 글로벌 후크가 고권한 프로세스에 주입 불가 → Youdao 우클릭 「관리자 권한으로 실행」.
  • 멀티모니터/고DPI에서 캡처 팝업이 안 보이는 곳으로 「날아감」: 화면별 DPI 배율 불일치 때문 → 캡처 설정에서 「창 위치 고정」으로 기본 화면 좌상단에 고정.
