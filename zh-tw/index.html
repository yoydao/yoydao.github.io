有道詞典 螢幕取詞 / 劃詞 / 截圖翻譯 / 快捷鍵 失效速查

有道詞典的螢幕取詞（滑鼠懸停查詞）、劃詞翻譯（選取即譯）、截圖 OCR 或快捷鍵突然不工作？先想清三件事——是沒啟用 / 沒給權限，還是某個程式（Office / PDF / 瀏覽器）相容問題，還是內容本身（掃描件 / 硬體加速）取不到可讀文字。輸入或點選你遇到的症狀或關鍵字（如 取詞失效、PDF取詞、劃詞、快捷鍵、64位、截圖），先看 L1 現象與官方定位，再看 L2 深度真因與排查，最後對照競品與真實情境。需要安裝 / 教學可到 有道詞典下載 / 取詞劃詞設定教學。

第一步：先分四類（L1·別一上來就重裝）

有道取詞 / 劃詞不工作，九成落在四類上、先分清誰的鍋：① 啟用 / 設定類——有道沒在背景跑（系統匣圖示灰了）、或設定裡沒勾「啟用取詞 / 劃詞 / 選取後自動翻譯」、取詞方式沒選對；② 權限類——macOS 沒給「輔助使用 + 螢幕錄製」、Windows 取詞目標是提權程式而有道沒以系統管理員執行；③ 程式相容類——Word/Excel 的有道 COM 增益集被停用、Acrobat 保護模式擋注入、瀏覽器沒裝取詞擴充；④ 渲染 / 內容類——掃描件 PDF 是圖片要走 OCR、瀏覽器/PDF 開了 GPU 硬體加速把文字畫成紋理、多螢幕 DPI 不一致彈窗飛走。更完整的設定見 有道詞典取詞劃詞設定教學。

啟用 / 設定類取詞 · 劃詞 · 自動翻譯 · OCR強力取詞
先確認有道在背景執行（系統匣圖示非灰），到 設定 → 取詞劃詞 勾選「啟用取詞」「啟用劃詞」「選取文字後自動翻譯」，並選對取詞方式（滑鼠懸停 / 按 Ctrl）。
權限類macOS 輔助使用+螢幕錄製 · Windows 系統管理員/UAC
macOS：系統設定 → 隱私權與安全性，給有道勾「輔助使用」「螢幕錄製」；Windows：取詞目標是「以系統管理員執行」的程式時，有道也要以系統管理員身分執行（同權限層級掛勾才注入得進去）。
程式相容類Office COM · Acrobat 保護模式 · 64位 · 瀏覽器擴充
Office：檔案 → 選項 → 增益集 → COM 增益集，勾選啟用有道增益集；Acrobat：關「啟動時啟用保護模式」，必要時把 YodaoDict.api 拷進 plug_ins；瀏覽器：裝有道取詞擴充。
快捷鍵類衝突 · 自訂 · 截圖OCR 熱鍵
設定 → 快捷鍵，看是否有衝突提示；改成 Ctrl+Shift / Alt+字母 這類不易撞的組合；截圖 OCR 單獨設熱鍵（如 Ctrl+Alt+O，避開其它軟體）。
渲染 / 顯示類硬體加速 · 掃描件 · DPI多螢幕 · 全螢幕
瀏覽器 / PDF 用 GPU 硬體加速會把內容渲成紋理、可見區無可讀文字（關硬體加速）；掃描件 PDF 是圖片要開 OCR；多螢幕彈窗飛走勾「固定視窗位置」。
網路 / 安裝類取詞速度 · 綠色版 · 增益集未載入
連線取詞慢先查網路；務必裝官方完整版——非官方「綠色版 / 精簡版」缺少註冊的取詞掛勾 DLL，PDF / 全域取詞會直接失效。

第二步：最常被忽略的真因 → L2 深度（螢幕取詞 = 全域掛勾 / DLL 注入）

關鍵認知：螢幕取詞不是「讀螢幕影像」，而是有道往別的程式裡注入一段取詞程式（全域掛勾 / DLL）去讀它的文字緩衝區。這就解釋了大半「玄學」失效——① 權限層級：一般權限的有道注不進「以系統管理員執行」的程式（反之亦然），所以某個程式裡取不到詞，多半是兩邊權限不一致；② 位元：32/64 位掛勾不互通，舊版有道對 64 位 Acrobat 會報「不相容的增效模組，無法載入 YodaoDict.api」；③ 沙箱：Acrobat「保護模式」是沙箱，直接擋第三方 .api 注入；④ 硬體加速：瀏覽器 / PDF 用 GPU 把內容渲成紋理後，可見區已無「可讀文字」，取詞自然讀不到；⑤ 掃描件：圖片型 PDF 根本沒有文字層，得用「OCR 強力取詞」。下圖是取詞啟用自檢面板示意；再往下是競品橫向對照與 2026 估算。

真實情境速判

  • 瀏覽器裡取不了詞、記事本 / Word 卻能取：多半是瀏覽器沒裝有道取詞擴充，或開了 GPU 硬體加速把網頁渲成紋理 → 裝擴充，或在瀏覽器關「使用硬體加速」後重啟。
  • Acrobat 64 位開啟 PDF 劃詞沒反應、提示「不相容的增效模組，無法載入 YodaoDict.api」：關 Acrobat「編輯 → 偏好設定 → 安全性(增強) → 啟動時啟用保護模式」，先開有道再開 Acrobat；仍不行把有道安裝目錄的 YodaoDict.api 拷進 Acrobat 的 plug_ins。
  • PDF 文字能選取卻取不到、或整頁都取不了：要麼是掃描件（圖片型 PDF，無文字層，得開「OCR 強力取詞」），要麼 Acrobat 保護模式 / Autotray.exe 占用擋了注入。
  • 新裝 Office（尤其教育版 / 365）後 Word 不能劃句翻譯：有道 Office COM 增益集被停用 → 檔案 → 選項 → 增益集 → COM 增益集 → 勾選啟用；不行就重裝有道修復增益集註冊。
  • 升級系統後 macOS 上取詞、熱鍵全失效：系統更新會重置權限 → 重新到「隱私權與安全性」給有道勾「輔助使用」「螢幕錄製」並重啟有道。
  • 取詞在一般程式好用、在某個「以系統管理員執行」的程式裡失靈：權限層級不同，全域掛勾注不進高權限程序 → 右鍵有道「以系統管理員身分執行」。
  • 多螢幕 / 高解析取詞彈窗「飛」到看不見的地方：各螢幕 DPI 縮放不一致所致 → 取詞設定裡勾「固定視窗位置」，把彈窗釘到主螢幕左上角。
