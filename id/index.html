Youdao tangkap kata di layar / terjemah seleksi / OCR / pintasan - pencari solusi

Apakah tangkap kata di layar Youdao (cari dengan arahkan kursor), terjemah seleksi, OCR, atau pintasan tiba-tiba berhenti? Pahami tiga hal dulu: apakah tidak diaktifkan / tanpa izin, kompatibilitas aplikasi tertentu (Office / PDF / browser), atau konten itu sendiri (pindaian / akselerasi perangkat keras) tanpa teks terbaca? Cari gejala atau kata kunci yang Anda alami (mis. tangkap, pdf, seleksi, pintasan, 64-bit, OCR): pertama gejala L1 dan posisi resmi, lalu akar penyebab L2 dan solusi, terakhir perbandingan rival dan kasus nyata. Instal / atur? Lihat unduh Youdao / panduan tangkap dan seleksi.

Langkah 1: bagi jadi empat kelompok dulu (L1 - jangan langsung instal ulang)

Saat tangkap/seleksi Youdao berhenti, 9 dari 10 kali itu salah satu dari empat kelompok - cari dulu siapa yang salah: ① aktivasi/pengaturan — Youdao tidak jalan di latar (ikon tray abu-abu), atau Anda tidak mencentang «aktifkan tangkap / seleksi / terjemah saat seleksi», atau pemicu salah; ② izin — macOS «Aksesibilitas + Perekaman Layar» belum diberikan, atau di Windows target aplikasi tinggi sedang Youdao tidak jalan sebagai administrator; ③ kompatibilitas aplikasi — add-in COM Youdao dinonaktifkan di Word/Excel, mode terlindungi Acrobat memblokir injeksi, atau browser tanpa ekstensi tangkap; ④ rendering/konten — PDF pindaian adalah gambar (butuh OCR), browser/PDF dengan akselerasi perangkat keras GPU menggambar teks jadi tekstur, atau DPI multi-monitor tak cocok melempar popup ke luar layar. Pengaturan lengkap: panduan tangkap dan seleksi Youdao.

Aktivasi / pengaturantangkap · seleksi · terjemah otomatis · OCR kuat
Pastikan Youdao jalan di latar (ikon tray bukan abu-abu), buka Pengaturan → Tangkap dan seleksi dan centang «aktifkan tangkap», «aktifkan seleksi», «terjemah saat seleksi», dan pilih pemicu yang benar (arahkan kursor / tahan Ctrl).
IzinmacOS Aksesibilitas+Perekaman Layar · Windows administrator/UAC
macOS: Pengaturan Sistem → Privasi dan Keamanan, beri Youdao «Aksesibilitas» dan «Perekaman Layar»; Windows: bila target «dijalankan sebagai administrator», jalankan Youdao juga sebagai administrator (hook hanya menyuntik di level privilese yang sama).
Kompatibilitas aplikasiOffice COM · mode terlindungi Acrobat · 64-bit · ekstensi browser
Office: File → Opsi → Add-in → Add-in COM, centang add-in Youdao; Acrobat: matikan «Aktifkan mode terlindungi saat mulai», bila perlu salin YodaoDict.api ke plug_ins; browser: pasang ekstensi tangkap Youdao.
Pintasankonflik · kustom · tombol OCR tangkapan
Pengaturan → Pintasan, perhatikan peringatan konflik; ganti ke kombinasi jarang seperti Ctrl+Shift / Alt+huruf; beri OCR tangkapan tombol sendiri (mis. Ctrl+Alt+O, jauh dari aplikasi lain).
Rendering / tampilanakselerasi perangkat keras · pindaian · DPI multi-monitor · layar penuh
Browser/PDF dengan akselerasi perangkat keras GPU merender konten jadi tekstur tanpa teks terbaca (matikan akselerasi); PDF pindaian adalah gambar (pakai OCR); untuk popup di luar layar pada multi-monitor centang «kunci posisi jendela».
Jaringan / instalasikecepatan tangkap · versi portabel · add-in tak dimuat
Pencarian online lambat? periksa jaringan; selalu pasang versi resmi lengkap - versi «portabel/ringan» tak resmi tak punya DLL hook tangkap terdaftar, jadi tangkap PDF/global gagal begitu saja.

Langkah 2: akar penyebab paling terabaikan → L2 (tangkap layar = hook global / injeksi DLL)

Inti: tangkap layar tidak «membaca citra layar» - Youdao menyuntikkan kode tangkap (hook global / DLL) ke aplikasi lain untuk membaca buffer teksnya. Ini menjelaskan sebagian besar kegagalan «misterius»: ① level privilese — Youdao berhak normal tak bisa menyuntik ke aplikasi «dijalankan sebagai administrator» (dan sebaliknya), jadi «hanya aplikasi itu tak menangkap» biasanya berarti privilese tak cocok; ② bitness — hook 32/64-bit tak bercampur, jadi Youdao lama di Acrobat 64-bit memberi «plugin tidak kompatibel, tidak bisa memuat YodaoDict.api»; ③ sandbox — «mode terlindungi» Acrobat adalah sandbox yang memblokir injeksi .api pihak ketiga; ④ akselerasi perangkat keras — begitu browser/PDF merender konten jadi tekstur GPU, tak ada «teks terbaca» untuk ditangkap; ⑤ pindaian — PDF hanya gambar tanpa lapisan teks, pakai «OCR tangkap kuat». Kasus nyata - baca cepat

  • Tak menangkap di browser tapi Notepad / Word bisa: biasanya browser tak punya ekstensi tangkap Youdao, atau akselerasi perangkat keras GPU merender halaman jadi tekstur → pasang ekstensi, atau matikan «gunakan akselerasi perangkat keras» dan mulai ulang.
  • Acrobat 64-bit, seleksi di PDF tak ada respons, «plugin tidak kompatibel, tidak bisa memuat YodaoDict.api»: matikan di Acrobat Edit → Preferensi → Keamanan (lanjutan) → «Aktifkan mode terlindungi saat mulai», buka Youdao sebelum Acrobat; jika tetap, salin YodaoDict.api dari folder Youdao ke plug_ins Acrobat.
  • Teks PDF bisa diseleksi tapi tak tertangkap, atau seluruh halaman tidak: itu pindaian (PDF gambar tanpa lapisan teks, butuh «OCR tangkap kuat»), atau mode terlindungi Acrobat / kunci Autotray.exe menghalangi injeksi.
  • Setelah pasang Office (terutama Education / 365) Word tak menerjemahkan seleksi: add-in COM Office Youdao dinonaktifkan → File → Opsi → Add-in → Add-in COM → aktifkan; jika tidak, instal ulang Youdao untuk mendaftarkannya lagi.
  • Setelah pembaruan OS, tangkap dan pintasan mati di macOS: pembaruan mereset izin → beri ulang Youdao «Aksesibilitas» dan «Perekaman Layar» di Privasi dan Keamanan lalu mulai ulang.
  • Tangkap bekerja di aplikasi normal tapi gagal di aplikasi «dijalankan sebagai administrator»: level privilese berbeda menghalangi hook global menyuntik ke proses tinggi → klik kanan Youdao «Jalankan sebagai administrator».
  • Multi-monitor / DPI tinggi: popup tangkap «terbang» ke tempat tak terlihat: akibat penskalaan DPI per layar tak cocok → centang «kunci posisi jendela» di pengaturan tangkap untuk memakukannya di kiri atas layar utama.
