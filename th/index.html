Youdao จับคำบนหน้าจอ / เลือกแปล / OCR / ปุ่มลัด - ค้นหาวิธีแก้

การจับคำบนหน้าจอของ Youdao (ชี้เมาส์เพื่อค้น), เลือกแปล, OCR หรือ ปุ่มลัด หยุดทำงานกะทันหัน? แยกสามอย่างก่อน: ยังไม่เปิด / ไม่มีสิทธิ์, ความเข้ากันได้ของแอปหนึ่ง ๆ (Office / PDF / เบราว์เซอร์), หรือเนื้อหาเอง (สแกน / เร่งฮาร์ดแวร์) ไม่มีข้อความอ่านได้? ค้นหาอาการหรือคำสำคัญที่พบ (เช่น จับ, pdf, เลือก, ปุ่มลัด, 64 บิต, OCR): ก่อนอื่น อาการ L1 และตำแหน่งทางการ, จากนั้น สาเหตุราก L2 และวิธีแก้, สุดท้ายเปรียบเทียบคู่แข่งและกรณีจริง. ติดตั้ง / ตั้งค่า? ดู ดาวน์โหลด Youdao / คู่มือจับและเลือก.

ขั้นที่ 1: แบ่งเป็นสี่กลุ่มก่อน (L1 - อย่าเพิ่งติดตั้งใหม่)

เมื่อจับ/เลือกของ Youdao หยุด 9 ใน 10 ครั้งเป็นหนึ่งในสี่กลุ่ม - หาก่อนว่าใครผิด: ① การเปิด/ตั้งค่า — Youdao ไม่ทำงานเบื้องหลัง (ไอคอนถาดเป็นสีเทา) หรือไม่ติ๊ก «เปิดจับ / เลือก / แปลอัตโนมัติเมื่อเลือก» หรือทริกเกอร์ผิด; ② สิทธิ์ — macOS ยังไม่ให้ «การช่วยการเข้าถึง + บันทึกหน้าจอ» หรือบน Windows เป้าหมายเป็นแอปสิทธิ์สูงแต่ Youdao ไม่ได้รันแบบผู้ดูแล; ③ ความเข้ากันได้ของแอป — ส่วนเสริม COM ของ Youdao ถูกปิดใน Word/Excel, โหมดป้องกัน Acrobat ขวางการฉีด, หรือเบราว์เซอร์ไม่มีส่วนขยายจับ; ④ การเรนเดอร์/เนื้อหา — PDF สแกนเป็นรูปภาพ (ต้อง OCR), เบราว์เซอร์/PDF เร่งฮาร์ดแวร์ GPU วาดข้อความเป็นพื้นผิว, หรือ DPI หลายจอไม่ตรงทำให้ป๊อปอัปลอยออกนอกจอ. การตั้งค่าเต็ม: คู่มือจับและเลือก Youdao.

การเปิด / ตั้งค่าจับ · เลือก · แปลอัตโนมัติ · OCR แบบแรง
ให้แน่ใจว่า Youdao ทำงานเบื้องหลัง (ไอคอนถาดไม่ใช่สีเทา) ไปที่ ตั้งค่า → จับและเลือก แล้วติ๊ก «เปิดจับ» «เปิดเลือก» «แปลอัตโนมัติเมื่อเลือก» และเลือกทริกเกอร์ที่ถูก (ชี้เมาส์ / กด Ctrl).
สิทธิ์macOS การช่วยการเข้าถึง+บันทึกหน้าจอ · Windows ผู้ดูแล/UAC
macOS: ตั้งค่าระบบ → ความเป็นส่วนตัวและความปลอดภัย ให้ Youdao «การช่วยการเข้าถึง» และ «บันทึกหน้าจอ»; Windows: เมื่อเป้าหมาย «รันแบบผู้ดูแล» ให้รัน Youdao แบบผู้ดูแลด้วย (ฮุกฉีดได้เฉพาะระดับสิทธิ์เดียวกัน).
ความเข้ากันได้ของแอปOffice COM · โหมดป้องกัน Acrobat · 64 บิต · ส่วนขยายเบราว์เซอร์
Office: ไฟล์ → ตัวเลือก → ส่วนเสริม → ส่วนเสริม COM ติ๊กส่วนเสริม Youdao; Acrobat: ปิด «เปิดโหมดป้องกันตอนเริ่ม» ถ้าจำเป็นคัดลอก YodaoDict.api ไป plug_ins; เบราว์เซอร์: ติดตั้งส่วนขยายจับ Youdao.
ปุ่มลัดชน · กำหนดเอง · ปุ่ม OCR ภาพหน้าจอ
ตั้งค่า → ปุ่มลัด ดูการเตือนชน; เปลี่ยนเป็นชุดที่ชนยากเช่น Ctrl+Shift / Alt+ตัวอักษร; ให้ OCR ภาพหน้าจอมีปุ่มของตัวเอง (เช่น Ctrl+Alt+O เลี่ยงแอปอื่น).
การเรนเดอร์ / แสดงผลเร่งฮาร์ดแวร์ · สแกน · DPI หลายจอ · เต็มจอ
เบราว์เซอร์/PDF เร่งฮาร์ดแวร์ GPU เรนเดอร์เนื้อหาเป็นพื้นผิวไม่มีข้อความอ่านได้ (ปิดการเร่ง); PDF สแกนเป็นรูป (ใช้ OCR); ป๊อปอัปนอกจอเมื่อหลายจอให้ติ๊ก «ตรึงตำแหน่งหน้าต่าง».
เครือข่าย / การติดตั้งความเร็วจับ · เวอร์ชันพกพา · ส่วนเสริมไม่โหลด
ค้นออนไลน์ช้า? ตรวจเครือข่าย; ติดตั้งเวอร์ชันทางการเต็มเสมอ - เวอร์ชัน «พกพา/บาง» ที่ไม่ทางการไม่มี DLL ฮุกจับที่ลงทะเบียน จึงจับ PDF/ส่วนกลางล้มเหลวทันที.

ขั้นที่ 2: สาเหตุรากที่มองข้ามมากที่สุด → L2 (จับหน้าจอ = ฮุกส่วนกลาง / ฉีด DLL)

แนวคิดหลัก: การจับหน้าจอไม่ได้ «อ่านภาพหน้าจอ» - Youdao ฉีดโค้ดจับ (ฮุกส่วนกลาง / DLL) เข้าไปในแอปอื่นเพื่ออ่านบัฟเฟอร์ข้อความของมัน. นี่อธิบายความล้มเหลว «ลึกลับ» ส่วนใหญ่: ① ระดับสิทธิ์ — Youdao สิทธิ์ปกติฉีดเข้าแอปที่ «รันแบบผู้ดูแล» ไม่ได้ (และกลับกัน) ดังนั้น «เฉพาะแอปนั้นจับไม่ได้» มักหมายถึงสิทธิ์ไม่ตรงกัน; ② บิต — ฮุก 32/64 บิตไม่ผสมกัน Youdao เก่าบน Acrobat 64 บิตจึงแจ้ง «ปลั๊กอินไม่เข้ากัน โหลด YodaoDict.api ไม่ได้»; ③ แซนด์บ็อกซ์ — «โหมดป้องกัน» ของ Acrobat คือแซนด์บ็อกซ์ที่ขวางการฉีด .api ของบุคคลที่สาม; ④ การเร่งฮาร์ดแวร์ — เมื่อเบราว์เซอร์/PDF เรนเดอร์เนื้อหาเป็นพื้นผิว GPU จะไม่เหลือ «ข้อความอ่านได้» ให้จับ; ⑤ สแกน — PDF ที่เป็นรูปอย่างเดียวไม่มีชั้นข้อความ ใช้ «OCR จับแบบแรง». ด้านล่างคือแผงตรวจสอบตนเอง; ถัดไปคือเปรียบเทียบคู่แข่งและประมาณการ 2026.

ตารางอาการครบ (อาการ L1 / ตำแหน่งทางการ · สาเหตุราก L2)

เปรียบเทียบเครื่องมือเลือก/จับของคู่แข่ง (โหมดจับ / OCR / แพลตฟอร์ม / ราคา · ข้อมูลสาธารณะ)

ซอฟต์แวร์จับหน้าจอ / เลือกOCR ภาพหน้าจอแพลตฟอร์มพจนานุกรมเสริมราคา (ประมาณ)
Youdaoจับส่วนกลาง + เลือกแปลOCR ภาพหน้าจอ + จับแบบแรงWin / Mac / iOS / Android / Web / ส่วนขยายเบราว์เซอร์mdx อ่อนหลักฟรี (บางสมาชิก / ไม่มีโฆษณา)
Eudicเลือก + จับหน้าจอจับภาพหน้าจอWin / Mac / iOS / Androidmdx แข็งแรง (คลังใหญ่)พื้นฐานฟรี + ขั้นสูงเสียเงิน
iCIBA (Kingsoft)จับ + เลือกแปลภาพหน้าจอWin / Mac / มือถือ / Webปานกลางฟรี (โฆษณา) + สมาชิก
Saladictเลือกในเบราว์เซอร์ (ส่วนขยาย)ขึ้นกับหน้า / OCR ต้องตั้งค่าส่วนขยายเบราว์เซอร์ (Chrome / Edge / Firefox)แหล่งกำหนดเองได้ฟรี · โอเพนซอร์ส
Microsoft Translator / Bingเลือกระดับระบบอ่อนOffice ในตัว + ภาพหน้าจอWin / Office / Edge / มือถือไม่มีฟรี

สาเหตุความล้มเหลวจับ Youdao และเปรียบเทียบเครื่องมือเลือก (ประมาณการ 2026)

ต่อไปนี้คือ ประมาณการ 2026 ที่สังเคราะห์จากความช่วยเหลือ / FAQ ทางการของ Youdao เอกสารสาธารณะของคู่แข่ง และการแก้ปัญหาของนักพัฒนา (ไม่ใช่คำมั่นของผู้ผลิต ไม่ใช่การวัดตรง; เพื่ออ้างอิง เปลี่ยนตามเวอร์ชันและ OS):

มิติประมาณการ / เปรียบเทียบ
สัดส่วนสาเหตุความล้มเหลวจับ (ชุมชน · ประมาณ)ไม่เปิด / สิทธิ์ ~40% > ความเข้ากันได้ของแอป (Office/PDF/เบราว์เซอร์) ~25% > ปุ่มลัดชน ~20% > การเรนเดอร์ / เนื้อหา (เร่งฮาร์ด / สแกน) ~15%
จับจริง ๆ คืออะไรจับหน้าจอ = ฮุกส่วนกลาง / ฉีด DLL ที่อ่านบัฟเฟอร์ข้อความของแอปเป้าหมาย (ไม่ได้อ่านภาพหน้าจอ); จึงได้รับผลจากระดับสิทธิ์ บิต แซนด์บ็อกซ์ การเร่งฮาร์ดแวร์
จับตามแพลตฟอร์ม (ประมาณ)เดสก์ท็อป (Win/Mac) รองรับ จับหน้าจอส่วนกลาง + เลือก + OCR ภาพหน้าจอ; มือถือส่วนใหญ่เลือก / แปลภาพถ่าย; เว็บ / เบราว์เซอร์ต้องมี ส่วนขยายจับ
ภาพรวมเครื่องมือเลือก / จับ (ประมาณ)Youdao = ทุกแพลตฟอร์ม + แปล AI แข็งแรง; Eudic = คลังเสริมแข็งแรง (mdx); iCIBA = ฟรีมวลชน; Saladict = เบราว์เซอร์โอเพนซอร์ส; Microsoft = Office ในตัว
ระดับราคา (ประมาณ)Youdao / iCIBA หลักฟรี (สมาชิกเพื่อไม่มีโฆษณา / เสริม) · Eudic ขั้นสูงเสียเงิน · Saladict ฟรีโอเพนซอร์ส · Microsoft Translator ฟรี

พื้นฐานประมาณการ: เส้นฐานแหล่งข้อมูล + การคาดการณ์ตามเวลา (ความช่วยเหลือ / FAQ ทางการ cidian.youdao.com, เอกสารสาธารณะ Eudic / iCIBA / Saladict / Microsoft, บทความนักพัฒนาสาธารณะ, 2026); เปลี่ยนตามเวอร์ชันและนโยบาย OS ของแต่ละผู้ให้บริการ. อ้างอิงข้อมูลทางการล่าสุด. โลคัลไลซ์ด้วย LLM.

กรณีจริง - อ่านเร็ว

  • จับในเบราว์เซอร์ไม่ได้แต่ Notepad / Word ได้: มักเบราว์เซอร์ไม่มีส่วนขยายจับ Youdao หรือเร่งฮาร์ดแวร์ GPU เรนเดอร์หน้าเป็นพื้นผิว → ติดตั้งส่วนขยาย หรือปิด «ใช้การเร่งฮาร์ดแวร์» แล้วเริ่มใหม่.
  • Acrobat 64 บิต เลือกใน PDF ไม่ตอบสนอง «ปลั๊กอินไม่เข้ากัน โหลด YodaoDict.api ไม่ได้»: ปิดใน Acrobat แก้ไข → การตั้งค่า → ความปลอดภัย (ขั้นสูง) → «เปิดโหมดป้องกันตอนเริ่ม» เปิด Youdao ก่อน Acrobat; ถ้ายัง คัดลอก YodaoDict.api จากโฟลเดอร์ Youdao ไป plug_ins ของ Acrobat.
  • ข้อความ PDF เลือกได้แต่จับไม่ได้ หรือทั้งหน้าไม่ได้: เป็นสแกน (PDF รูปไม่มีชั้นข้อความ ต้อง «OCR จับแบบแรง») หรือโหมดป้องกัน Acrobat / การล็อก Autotray.exe ขวางการฉีด.
  • หลังติดตั้ง Office (โดยเฉพาะ Education / 365) Word แปลข้อความที่เลือกไม่ได้: ส่วนเสริม COM Office ของ Youdao ถูกปิด → ไฟล์ → ตัวเลือก → ส่วนเสริม → ส่วนเสริม COM → เปิด; ถ้าไม่ได้ ติดตั้ง Youdao ใหม่เพื่อลงทะเบียนใหม่.
  • หลังอัปเดต OS จับและปุ่มลัดตายบน macOS: อัปเดตรีเซ็ตสิทธิ์ → ให้ Youdao «การช่วยการเข้าถึง» และ «บันทึกหน้าจอ» ใหม่ในความเป็นส่วนตัวและความปลอดภัย แล้วเริ่มใหม่.
  • จับได้ในแอปปกติแต่ล้มเหลวในแอป «รันแบบผู้ดูแล»: ระดับสิทธิ์ต่างกันทำให้ฮุกส่วนกลางฉีดเข้ากระบวนการสิทธิ์สูงไม่ได้ → คลิกขวา Youdao «รันในฐานะผู้ดูแล».
  • หลายจอ / DPI สูง: ป๊อปอัปจับ «ลอย» ไปที่มองไม่เห็น: เพราะสเกล DPI แต่ละจอไม่ตรง → ติ๊ก «ตรึงตำแหน่งหน้าต่าง» ในตั้งค่าจับเพื่อตรึงที่มุมบนซ้ายของจอหลัก.
